# HWP/HWPX 변환기 v8.0 (PyQt6)

한글(HWP/HWPX) 파일을 PDF, HWPX, 또는 **DOCX**로 일괄 변환하는 Windows용 현대적 GUI 프로그램

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![PyQt6](https://img.shields.io/badge/PyQt6-6.0+-green.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## ✨ 주요 기능

### 🎯 핵심 기능
- **폴더 일괄 변환**: 폴더 내 모든 HWP/HWPX 파일을 한 번에 변환
- **파일 개별 선택**: 원하는 파일만 골라서 변환
- **다양한 변환 형식**: PDF, HWPX, **DOCX** (v8.0 신규) 지원
- **하위 폴더 포함**: 하위 폴더의 파일도 자동으로 변환
- **덮어쓰기 방지**: 기존 파일 보호 옵션 (자동 번호 추가)

### 🎨 UI/UX
- **다크/라이트 테마**: 🌙/☀️ 버튼으로 즉시 전환
- **드래그 앤 드롭**: 파일을 끌어다 놓기만 하면 추가
- **현대적 디자인**: 그라데이션, 애니메이션, 아이콘 적용
- **Toast 알림**: 비차단 토스트 팝업 (v8.0 신규)
- **HiDPI 지원**: 고해상도 디스플레이 자동 스케일링 (v8.0 신규)

### 💡 편의 기능
- **설정 자동 저장**: 마지막 선택 폴더 및 옵션 기억
- **변환 취소**: 진행 중인 변환 작업 중단 가능
- **진행률 표시**: 실시간 진행 상황 + **예상 남은 시간** (v8.0 신규)
- **결과 보고서**: 성공/실패 파일 상세 정보 및 소요 시간 제공

## 💻 시스템 요구사항

## 📦 설치 방법

### 1. Python 및 의존성 설치
```bash
pip install pywin32 PyQt6
```

### 2. 실행
```bash
# 관리자 권한으로 실행 (필수!)
python hwptopdf-hwpx_v4.py
```

### 3. 빌드 (선택)
```bash
# PyInstaller로 단일 실행 파일 생성
pyinstaller hwp_converter.spec
```

## 🛠️ v8.0 변경사항

### 신규 기능
- ✅ **DOCX 변환 지원**: HWP → DOCX 직접 변환
- ✅ **Toast 알림**: 페이드 애니메이션 토스트 팝업
- ✅ **HiDPI 지원**: 고해상도 디스플레이 자동 스케일링
- ✅ **예상 시간 표시**: 변환 진행 중 남은 시간 표시
- ✅ **소요 시간 통계**: 변환 완료 시 총 소요 시간 표시

### 코드 개선
- 상수 중앙화 (VERSION, FORMAT_TYPES)
- 타입 힌트 보완
- 메모리 누수 방지 강화
